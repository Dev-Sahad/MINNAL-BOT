🐙 GitHub Sync

Pushes your current settings.json and panel to github.com/Dev-Sahad/MINNAL-BOT — Railway auto-deploys on every push.
⚠️ GITHUB_PERSONAL_ACCESS_TOKEN is not set in Railway. Follow these steps:
1Open Railway → MINNAL THE GOAT project
2Click your MINNAL-BOT service → Variables tab
3Click New Variable and add:
Name: GITHUB_PERSONAL_ACCESS_TOKEN
Value: (paste your GitHub PAT here)
4Click Add — Railway will redeploy automatically ⚡

⚙️ How It Works

1. Edit any setting in this panel and click Save
2. Click Push to GitHub above

4. Railway detects the push and auto-redeploys the bot
5. Your bot is live with the new settings! ⚡
